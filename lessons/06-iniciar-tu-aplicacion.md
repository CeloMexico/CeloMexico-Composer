# Lección 6: Iniciar tu aplicación

## Objetivo
Levantar el servidor de desarrollo y ver la app en acción.

## Comandos

```bash
yarn dev
```

O directamente:

```bash
yarn workspace @examples/react-app dev
```

Luego abre en el navegador:

[http://localhost:3000](http://localhost:3000)
